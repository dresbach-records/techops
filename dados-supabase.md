# OBSOLETO - Conexão Direta com Supabase Removida

Este arquivo continha chaves e exemplos de conexão direta do frontend com o Supabase.

De acordo com a arquitetura final definida, **o frontend NUNCA deve se conectar diretamente ao Supabase**. Toda a comunicação com o banco de dados deve ser feita exclusivamente através do backend em Go.

Este arquivo foi esvaziado para reforçar essa regra e prevenir o uso indevido de conexões diretas.
